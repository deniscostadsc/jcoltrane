Conhecimento Prévio Necessário

Esse guia foi feito para usuários interessados em usar o JColtrane em sua aplicação. É suposto que o leitor tenha conhecimento básico sobre JAVA Annotation (saber como anotações para métodos e parâmetros funcionam é suficiente). Assim, se recomenda a leitura de documentação específica se você não é familiar à esses conceitos.

Uma vez que o JColtraine é usado para análise de documentos XML no JAVA, é assumido, logicamente, que o usuário sabe XML e a linguagem JAVA.

Embora JColtrane use SAX, não é necessario conhecimento prévio sobre o mesmo para entender esse guia. Contudo, usuários que conhecem o SAX entenderão profundamente as comparações entre as abordagens e chegarão às suas próprias conclusões.

Dessa forma, se você está seguro com relação aos tópicos anteriores, vá para o próximo conteúdo.
